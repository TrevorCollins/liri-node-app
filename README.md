# LIRI (or Language Integrated Response Interface)

LIRI is a CLI app that takes in simple commands and returns parsed info from various api's.

## APIs Used

* Spotify
* Axios
    * OMDB
    * Bands In Town
* FS
* Moment
* Dotenv
